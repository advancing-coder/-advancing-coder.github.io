回溯算法-力扣 77组合

题目描述

给定两个整数 n 和 k，返回 1 … n 中所有可能的 k 个数的组合。

示例:

输入: n = 4, k = 2
输出:
[
[2,4],
[3,4],
[2,3],
[1,2],
[1,3],
[1,4],
]

AC代码

总结

回溯算法大概就是遍历决策树的过程,把所有的可能都遍历完,我的理解是遍历树的话应该有两种方式,一种dfs是利用递归的方式,如果利用非递归也是利用栈,区别是一个利用系统栈,另一种是自己建栈。另一种是bfs是利用队列来实现

感觉回溯算法的技巧大概是你先遍历你这个数组,也就是第一次决策所有的可能,
然后是做出决策,再在这个决策的基础上递归调用,执行下一个决策,当决策不满足时,再撤销决策
