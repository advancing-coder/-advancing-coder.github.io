力扣445两数相加

题目描述

给你两个 非空 链表来代表两个非负整数。数字最高位位于链表开始位置。它们的每个节点只存储一位数字。将这两数相加会返回一个新的链表。

你可以假设除了数字 0 之外，这两个数字都不会以零开头。

实例

输入：
(7 -> 2 -> 4 -> 3) + (5 -> 6 ->4)
输出：7 -> 8 -> 0 -> 7

解题思路

把两个链表反转然后做加法,再把和反转一下就得到结果

主要是在做加法的时候对于进位的一些细节处理，以及链表没有头节点的处理问题，个人感觉在自己做的时候还是把链表设置一个头节点比较舒服，因为题目给出了没有头节点所以在处理的时候有一些细节的体现

还有就是在链表反转函数的时候注意一下return的细节

代码解析

在链表相加函数中,是把结果串在l1上的，所以先记录他的头节点
当l1和l2都非空的时候我们把两个相加只需更改l1的val值
进位需要考虑l1和l2是否为空的情况是否需要malloc一个新的节点这些细节都是非常重要的。

踩过的一个坑就是c++ malloc一段内存必须要释放出来，不然要报错，因为没有仔细看报错的具体信息，一直以为是用了空指针，造成了很多麻烦，还有就是涉及指针的调试都不太会，不会查看内存窗口。
